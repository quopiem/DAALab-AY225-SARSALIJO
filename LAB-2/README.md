# Sorting Algorithm Benchmarking Tool

## Laboratory 2: Multi-Algorithm Benchmarking

This application is a **GUI-based** benchmarking tool for sorting algorithms, developed separately from the Console implementation.

### Features
- **Algorithms**: Bubble Sort, Insertion Sort, Merge Sort.
- **Interface**: Graphical User Interface (Tkinter).
- **Dynamic Input**: User can specify dataset sizes (e.g., 10,000, 20,000).
- **Verification**: Automatic verification of sorted output.
- **Timing**: Precise execution timing excluding data generation.

### Prerequisites
- Python 3.x
- Tkinter (included with standard Python installation)

### How to Run
```bash
python app.py
```

### Usage
1. Enter the desired dataset size (e.g., 5000).
2. Select an algorithm from the dropdown.
3. Click "START SORTING" to generate random data and sort it.
4. View results and timing statistics.
