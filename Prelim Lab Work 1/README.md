# Bubble Sort Empirical Analysis

This project implements the Bubble Sort algorithm in Python to empirically analyze its performance on a dataset of integers.

## Description

The script reads integers from a `dataset.txt` file, sorts them using an optimized Bubble Sort algorithm, and outputs the execution time and the sorted array.

## Requirements

*   Python 3.x
*   `dataset.txt` file containing integers (one per line) in the same directory.

## How to Run

1.  Ensure `bubble_sort.py` and `dataset.txt` are in the same folder.
2.  Open a terminal or command prompt in that folder.
3.  Run the script using:
    ```bash
    python bubble_sort.py
    ```

## Output

The program will display:
*   Status messages (reading file, sorting).
*   Execution time in seconds.
*   The full sorted array.
